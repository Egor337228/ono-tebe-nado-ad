# Оно тебе надо — АД

Учебный проект, посвящённый теме «Оно тебе надо» в необычном прочтении — «АД».

## 📖 О проекте

Этот проект является тренировочным заданием в рамках обучения вёрстке. Основная цель — сверстать страницу по заданному макету, отрабатывая навыки работы с HTML и CSS.

Проект создан на основе шаблона от [Яндекс.Практикума](https://github.com/yandex-praktikum/ono-tebe-nado-ad).

## 🛠 Технологии

- **HTML** — 33.7%
- **CSS** — 66.3%

Файловая структура:
- `index.html` — основной HTML-файл
- `styles/` — CSS-стили
- `fonts/` — шрифты
- `images/` — изображения
- `.github/workflows/` — CI/CD конфигурация
